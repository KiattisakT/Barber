# Owner Questions

คำถามเหล่านี้ใช้คุยกับเจ้าของร้าน Dream Catcher ก่อนเริ่มออกแบบหน้าจอหรือ implement

## Shop Basics

1. ชื่อร้านที่ต้องการแสดงบนหน้าจองคือ `Dream Catcher Barber and Tattoo` หรือมีชื่อภาษาไทยด้วยหรือไม่
2. เวลาทำการจริงคือ 10:00-20:00 ใช่หรือไม่
3. ร้านหยุดทุกวันพุธใช่หรือไม่
4. มีวันหยุดพิเศษหรือช่วงเวลาที่ไม่รับคิวเป็นประจำหรือไม่
5. ช่องทางติดต่อหลักที่อยากให้ลูกค้าเห็นคือเบอร์โทร, LINE, Facebook หรือ Google Maps

## Barber Services

1. บริการตัดผมมีอะไรบ้าง
2. แต่ละบริการราคาเท่าไหร่
3. แต่ละบริการใช้เวลากี่นาที
4. ต้องมี buffer ระหว่างคิวหรือไม่
5. รับจองล่วงหน้าได้กี่วัน
6. ลูกค้าควรยกเลิกหรือเลื่อนคิวเองได้ล่วงหน้ากี่ชั่วโมง

## Walk-In Rules

1. Walk-in สามารถแทรกระหว่าง online booking ได้หรือไม่
2. ถ้ามีลูกค้า online booking และ walk-in รอพร้อมกัน ร้านให้ priority อย่างไร
3. ต้องการให้ระบบประเมินเวลารอสำหรับ walk-in หรือไม่
4. ถ้าลูกค้า online มาช้า ร้านจะรอกี่นาทีก่อนถือว่า no-show

## Staff

1. ตอนนี้มีช่างกี่คน
2. ถ้าเพิ่มช่างอีกคน ลูกค้าเลือกช่างเองได้หรือร้าน assign เอง
3. ช่างแต่ละคนมีเวลาทำงานต่างกันหรือไม่
4. ช่างแต่ละคนทำได้ทั้งตัดผมและสักลายหรือแยกหน้าที่

## Tattoo Requests

1. ร้านต้องการถามข้อมูลอะไรจากลูกค้าก่อนประเมินงานสัก
2. ลูกค้าควรแนบรูป reference ได้กี่รูป
3. ต้องถามตำแหน่งบนร่างกายแบบ free text หรือมีตัวเลือก
4. ต้องถามขนาดเป็นเซนติเมตรหรือใช้ขนาดประมาณ เช่น เล็ก กลาง ใหญ่
5. ต้องถามงบประมาณหรือไม่
6. หลังลูกค้าส่ง request ร้านต้องการให้ข้อความตอบกลับบอกอะไร
7. งานสักบางแบบสามารถจองเวลาได้ทันทีหรือทุกงานต้อง review ก่อน

## Notifications

1. MVP แรกใช้แค่ confirmation page ก่อน หรืออยากให้มี SMS/LINE ตั้งแต่แรก
2. ถ้าใช้ LINE ลูกค้าควรกรอก LINE ID หรือให้แอด LINE OA
3. Reminder ควรส่งก่อนเวลานัดกี่ชั่วโมง

## Branding And Content

1. ร้านมี logo หรือรูปหน้าร้านไหม
2. โทนภาพที่ต้องการคือ barber classic, tattoo studio, minimal, dark, หรืออื่นๆ
3. มีรูปผลงานตัดผมหรือสักลายที่อยากใช้บนหน้าจองไหม
4. ต้องการภาษาไทยอย่างเดียว หรือไทย/อังกฤษ
